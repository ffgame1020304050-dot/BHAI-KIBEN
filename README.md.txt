# BHAI TOP UP - Free Fire Diamond Top Up Website

## Features
1. Complete Free Fire Diamond top-up website
2. WhatsApp notifications to 01748320647
3. Admin dashboard with login (password: 19923581)
4. Order management (confirm/cancel/delete)
5. Payment via Bkash/Nagad (01540651159)
6. Recent orders display
7. Responsive design

## Installation
1. Save all files in the same folder
2. Open `index.html` in a web browser
3. For admin access, go to `admin.html` or click Admin button

## Package Categories
1. Diamond Packages (25 to 10120 Diamonds)
2. Weekly & Monthly Packages
3. AirDrop Packages
4. Evo Access Packages

## How to Use
1. Customer selects a package
2. Enters Free Fire UID and phone number
3. Makes payment to 01540651159
4. Enters transaction ID and submits
5. Admin gets WhatsApp notification
6. Admin logs in to confirm/cancel order
7. Customer gets status update via WhatsApp

## Admin Access
- URL: `admin.html`
- Password: `19923581`
- Can confirm, cancel, or delete orders
- Can export orders as JSON
- Can clear completed orders
